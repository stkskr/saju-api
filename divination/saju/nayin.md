# Nayin (납음/納音) — 60 jiazi sound elements

Each pair of consecutive jiazi shares one of 30 nayin names. The nayin is a secondary element classification — distinct from the stem and branch elements — used in compatibility readings, classical horoscopy, and older Bazi schools.

## How to use

Two people's day pillar nayin can be checked for compatibility — generating relationships (e.g., metal-water) suggest support; controlling (metal-wood) suggest friction. Same nayin = mirroring temperament.
